# Aureon Music Group — Cinematic Website

Upload these files to the root of your `aureon-starter` project in GitHub, replacing the old files.

## Important
The homepage uses `/public/images/aureon-cinematic-hero.png` as the cinematic background.
To use a real looping video later, add `public/videos/studio-loop.mp4` and we can switch the hero to video.

All navigation links work and route to their screens.
