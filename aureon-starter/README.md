# Aureon Music Group Website Starter

A premium Next.js starter website for Aureon Music Group.

## Quick start

1. Install Node.js LTS.
2. Open this folder in VS Code.
3. Run:

```bash
npm install
npm run dev
```

4. Open http://localhost:3000

## Deploy on Vercel

1. Create a GitHub repository.
2. Upload this folder.
3. Go to Vercel and import the repository.
4. Add your domain: aureonmusicgroup.com

## Replace placeholders

- Put your looping background video here: `public/videos/studio-loop.mp4`
- Put cover art/images here: `public/images/`
- Update shop links in `app/page.tsx`
- Update contact email in `app/page.tsx`
