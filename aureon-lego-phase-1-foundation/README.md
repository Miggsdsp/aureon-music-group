# Aureon Music Group — Lego Phase 1

This package is the clean foundation. It is designed to replace the old starter site.

## Upload method
1. In GitHub, delete the old folders: `aureon-starter` and `aureon-phase-3-cinematic`.
2. Upload the CONTENTS of this zip to the repository root, not inside a folder.
3. In Vercel Project Settings, set Root Directory to `./`.
4. Redeploy.

## Video background
The homepage already uses the approved cinematic studio image as a fallback.
To add real movement later, upload your looping MP4 as:
`public/videos/studio-loop.mp4`

The site will automatically use it.
