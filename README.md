# Aureon Music Group — Real Phase 1

This is the clean production foundation for Aureon Music Group.

## What is included

- Next.js app-router project
- Cinematic black/gold homepage
- Working navigation pages
- Animated studio background fallback
- Video-ready hero section
- Artist, Music, Videos, Licensing, Merchandise and Contact pages
- SEO metadata
- Responsive mobile layout

## Important upload instructions

Do not upload this folder as a folder inside GitHub.

Open this folder and upload the contents to the root of the repository:

- app
- components
- public
- package.json
- next.config.mjs
- tsconfig.json
- next-env.d.ts
- README.md
- .gitignore

Vercel root directory should be:

./
